# DC101_Game_SargaLeslie

Rock–Paper–Scissors Game

-A lightweight and interactive Rock–Paper–Scissors game built with HTML, CSS, and JavaScript.
Play directly in the browser and enjoy instant results with clean UI and smooth gameplay.

Technologies Used

HTML5 – Game layout
CSS3 – Styling and visuals
JavaScript (ES6) – Core game logic

How to Play

1.Choose Rock, Paper, or Scissors.
2.The computer will randomly choose its move.
3.The result will appear instantly on screen.
4.Scores update after each round.
